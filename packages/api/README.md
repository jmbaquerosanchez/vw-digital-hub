# VW Digital Hub Design System

## Scripts

1.  **Run tests**

    ```bash
    npm run dest:dev
    ```

2.  **run linter**

    ```bash
    npm run lint
    ```

3.  **run dev mode**

    ```bash
    npm run start
    ```

4.  **run Storybook**

    ```bash
    npm run start:storybook
    ```

5.  **Build**
    ```bash
    npm run build
    ```
